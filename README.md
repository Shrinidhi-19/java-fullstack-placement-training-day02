# Day 2 - HTML Fundamentals Practice

## Overview

This repository contains the HTML programs completed during Day 2 of the Java Full Stack Development Placement Training.

The exercises focus on understanding the fundamentals of web development using HTML and demonstrate the creation of static web pages with various HTML elements.

---

## Programs Included

### 1. Tribute Page - Mahendra Singh Dhoni (MSD)

A simple tribute webpage dedicated to Mahendra Singh Dhoni featuring:

* Headings and paragraphs
* Text formatting tags
* Images
* Cricket match statistics table
* External links to videos
* Basic webpage structure

#### Concepts Practiced

* HTML document structure
* Headings (`<h1> - <h6>`)
* Paragraphs (`<p>`)
* Text formatting (`<i>`, `<b>`, `<u>`)
* Images (`<img>`)
* Tables (`<table>`)
* Hyperlinks (`<a>`)
* Horizontal rules (`<hr>`)

---

### 2. Shivaji: The Boss Fan Webpage

A movie-themed webpage showcasing different HTML elements.

#### Features

* Background color customization
* Hyperlinks
* Images
* Ordered and unordered lists
* Marquee text
* Audio embedding
* Tables
* User registration form
* Footer section

#### Concepts Practiced

* Page styling using HTML attributes
* Ordered Lists (`<ol>`)
* Unordered Lists (`<ul>`)
* Multimedia Elements (`<audio>`)
* Tables
* Forms and Input Fields
* Radio Buttons
* Checkboxes
* Buttons
* Footer Section

---

## Learning Outcomes

After completing these programs, the following concepts were understood:

* Structure of an HTML document
* Creating static webpages
* Adding images and hyperlinks
* Working with tables and lists
* Creating forms for user input
* Embedding multimedia content
* Organizing webpage content effectively
